# Diyagram Kaynakları

Bu klasör elle yönetilen UML kaynaklarını içerir.

- `UML.drawio`: Düzenlenebilir Draw.io kaynağı.
- `UML.jpg`: Paylaşım ve hızlı görüntüleme için dışa aktarılmış görsel.

Diyagramlar kod değişiklikleriyle otomatik güncellenmez. Diyagram ile kod arasında fark varsa [Güncel Sistem Mimarisi](../architecture/guncel_sistem_mimarisi.md) ve çalışan kod esas alınır.

Yeni sınıflar veya modül sınırları diyagrama elle işlendiğinde `.drawio` kaynağı ile dışa aktarılan görsel birlikte güncellenmelidir. Bu klasör mimari kararları açıklamak yerine görselleştirir; kararların gerekçesi [Mimari Belgeler](../architecture/README.md) altında tutulur.
