# Ayrilmis Demo Girdileri

Bu klasordeki her senaryo web arayuzunde uc ayri dosya olarak kullanilir:

1. `issue.json` -> Issue JSON
2. `metadata.json` -> Metadata JSON
3. `checklist.json` -> Checklist JSON

Dosyalar birlestirilmis `AnalyzeRequest` fixture'larindan farkli olarak gercek kullanicinin ayri kaynaklardan veri yuklemesini taklit eder.

Onerilen baslangic senaryosu `aud-002-missing-acceptance-criteria` klasorudur.
